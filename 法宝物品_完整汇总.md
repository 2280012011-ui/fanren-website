## 掌天瓶

| 属性 | 内容 |
|------|------|
| **类型** | 玄天之宝/仙器 |
| **品阶** | 玄天之宝/仙器 |
| **所属** | 韩立 |
| **初登场** | 第1章 |
| **能力** | 催熟灵药、时间法则、混沌遗存、绿液凝聚 |
| **获取方式** | 七玄门彩霞山偶然拾得 |

### 详细介绍

人界篇第一至宝，又名小绿瓶。外形为墨绿色古朴玉瓶，实为先天玄天至宝——天道初显时混沌所生的第一件玄天之物。瓶身坠入人界彩霞山，被韩立在七玄门时偶然捡到。核心能力：(1)吸收月光凝聚绿液，可大幅加速灵药生长，是韩立修炼突破的根本依仗；(2)完整形态可凝聚时间晶粒、穿越时空、改写因果；(3)内含混沌遗存，是击败终极BOSS古或今的关键。原为仙界九元观镇派之宝，被九元道祖所得，后因叛徒盗取失败而分离。瓶灵具有独立意识，可辅助战斗。

### 重要用途

韩立修炼突破的根本依仗，贯穿全书的核心外挂

|------|------|

### 重要用途

韩立本命法宝，伴随整个修仙生涯
